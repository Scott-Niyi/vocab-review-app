# 用户指南 - Bug修复说明

## 已修复的问题

### 1. 评分系统优化 ✅

**之前的问题**:
- Barely (2分) 给 -8分，导致熟悉度降低到3%
- 进步太慢，不够激励

**现在的改进**:
- **1 (Don't know)**: -20分 - 完全不认识
- **2 (Barely)**: +5分 - 勉强认识（现在是正向提升！）
- **3 (Familiar)**: +15分 - 有些熟悉
- **4 (Know well)**: +25分 - 很熟悉
- **5 (Master)**: +35分 - 完全掌握

**效果**: 现在即使选"Barely"也会有小幅进步，更符合学习曲线！

### 2. Review次数统计

**说明**: Review次数（timesReviewed）一直都在正常增加，每次评分都会+1。
- 在Library中评分也会增加review次数
- 在Review模式评分也会增加review次数
- 可以在Library的详情页看到"Reviewed X×"

### 3. Hyperlink跳转问题

**可能的原因**:
- 新添加的单词还没有加入到当前复习队列
- 需要重新开始session才能看到新单词

**解决方案**:
1. 添加新单词后，点击"Start New Session"重新加载队列
2. 或者在Library中直接查看和编辑新单词
3. Hyperlink会自动将目标单词添加到当前队列

### 4. 删除Image问题

**如果遇到重复词条**:
1. 检查是否点击了多次"Save"按钮
2. 刷新页面，重新加载数据
3. 如果仍有问题，请提供详细步骤以便复现
