## 🧪 如何测试导入功能

### 方法1：直接导入测试项目

1. 在 Electron 应用中，点击 **"📂 Open Existing Project"**
2. 在文件夹选择对话框中，导航到：
   ```
   ~/Library/Mobile Documents/com~apple~CloudDocs/英语单词/vocab-review-app/
   ```
3. 选择 **`test-vocab-project`** 文件夹
4. 点击"Open Project"
5. 应用会加载3个示例单词

### 方法2：复制到其他位置测试

你可以把 `test-vocab-project` 文件夹复制到任何地方：

```bash
# 复制到桌面
cp -r vocab-review-app/test-vocab-project ~/Desktop/

# 复制到 Documents
cp -r vocab-review-app/test-vocab-project ~/Documents/

# 复制到自定义位置
cp -r vocab-review-app/test-vocab-project ~/Documents/VocabProjects/
```

然后在应用中导入复制后的文件夹。

## 🆕 创建新项目测试

1. 点击 **"✨ Create New Project"**
2. 系统会打开保存对话框
3. 建议位置：`~/Documents/VocabProjects/`
4. 输入项目名称，例如：`My-GRE-Words`
5. 点击"Create Project"
6. 应用会创建空项目并加载 mock 数据

## 🔄 切换项目测试

1. 打开一个项目后
2. 点击右上角的**设置图标**（⚙️）
3. 在"Data Storage"部分，点击 **"📚 Switch Project"**
4. 确认保存
5. 应用返回项目选择界面
6. 选择另一个项目

## 📊 验证数据持久化

1. 打开测试项目
2. 添加一个新单词
3. 关闭应用
4. 在 Finder 中打开 `test-vocab-project/vocabulary.json`
5. 验证新单词已保存

## 🗂️ 在 Finder 中查看
